# RetainAI

AI-native retention intelligence platform and multicloud product foundation.

RetainAI is a public portfolio project focused on employee-retention analytics, explainability, MLOps practices, cost/security controls, and a multicloud deployment model.

It is presented here as a technical case study: product direction, architecture, infrastructure, documentation, and AI-assisted implementation.

## Current public version

- Source repository: [HubertRonald/RetainAI](https://github.com/HubertRonald/RetainAI)
- Current documentation snapshot: `v0.4.0-alpha.1`
- Dashboard target domain: `https://retainai.hubertronald.dev/`
- Backend target domain: `https://api.retainai.hubertronald.dev/`

## Positioning

RetainAI is not only a demo dashboard. It is a product-oriented architecture exercise covering:

- machine learning product design;
- explainability and retention-risk interpretation;
- Streamlit dashboard delivery;
- API-based backend delivery;
- GCP Cloud Run frontend deployment;
- AWS API Gateway and Lambda backend deployment;
- Terraform-based infrastructure;
- authentication and quota boundaries;
- future AI/RAG advisory capabilities.
