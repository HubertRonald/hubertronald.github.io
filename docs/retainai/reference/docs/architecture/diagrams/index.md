# RetainAI architecture visual sources

The final diagrams are generated with official AWS and Google Cloud product
icons rather than Mermaid service boxes.

## Commands

```bash
./scripts/download_architecture_assets.sh
./scripts/render_architecture_diagrams.sh
```

## Outputs

```text
figs/retainai_decision_intelligence_architecture.svg
figs/retainai_decision_intelligence_architecture.png
```

Solid styling means implemented or deployed. Dashed styling means planned,
disabled, or post-`v0.4`.
